## 📀 Como rodar esse projeto ?


```bash
# Clone esse repositório
$  git clone linkrepo

# Acesse a pasta do projeto no seu terminal
$ cd projetosena

#Instale as dependências
$ npm install

#Execute a aplicação
$ npm run start

# A aplicação será iniciada na porta 3000, acesse pelo navegador: http://localhost:3000
```
